## 📌 About

This is the personal portfolio of **Mohammed Shafiq AS**, a final-year B.Tech student in Computer Science & Business Systems at VSB Engineering College, Karur. The site is built as a single-page HTML application with a dark cyber-aesthetic design, featuring animated particle backgrounds, glassmorphism cards, and a custom cursor.

---

## ✨ Features

- **Animated particle canvas** background with interactive network lines
- **Custom cursor** with smooth ring-follow animation
- **Glassmorphism UI** — frosted-glass cards with hover effects
- **Smooth scroll navigation** with a sticky blur navbar
- **Fully responsive** — mobile hamburger menu included
- **Scroll reveal animations** for all sections
- **Contact form** wired to mailto
